# Audio Usage Notes

The four WAV files are original synthesized interface cues created for this task package.

- correct_answer.wav: positive three-note confirmation.
- incorrect_answer.wav: gentle two-note retry cue.
- level_complete.wav: short completion fanfare.
- button_click.wav: subtle interface click.

Technical format: mono WAV, 44.1 kHz, 16-bit PCM.

Use a visible Sound On or Sound Off control. Do not autoplay audio before the user interacts with the page. Keep audio volume gentle and consistent.
