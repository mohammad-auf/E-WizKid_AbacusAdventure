E-WIZ KID ABACUS ADVENTURE — ASSET LICENCE NOTES
============================================================

This file tracks the licence status of every third-party asset
used in the project (images, icons, audio, fonts).

IMPORTANT: All assets must be documented here BEFORE they are
added to the 05_Assets/ directory or any other project folder.
Using an unlicensed asset is a copyright violation.

------------------------------------------------------------

HOW TO COMPLETE THIS FILE
------------------------------------------------------------

When adding a new asset, add an entry using this template:

  File:       05_Assets/<folder>/<filename>
  Source:     URL where the asset was obtained
  Author:     Creator's name or studio
  Licence:    Full licence name (e.g., CC0, CC BY 4.0, OFL, MIT)
  Licence URL: Link to the full licence text
  Modifications: None | [describe any edits made]
  Phase Added: Phase number

------------------------------------------------------------

ACCEPTABLE LICENCES
------------------------------------------------------------

The following licence types are approved for use in this project:

  ✅ CC0 (Creative Commons Zero / Public Domain)
  ✅ CC BY 4.0 (Attribution required — credit in this file is sufficient)
  ✅ CC BY-SA 4.0 (Attribution + Share-Alike)
  ✅ MIT (for code-based assets / icon libraries)
  ✅ Apache 2.0
  ✅ Open Font Licence (OFL) — for typefaces only
  ✅ Royalty-Free (confirmed, with source URL)
  ✅ Custom permission (written confirmation required)

The following licence types are NOT approved:
  ❌ CC BY-NC (Non-Commercial only — not suitable if project is commercial)
  ❌ CC BY-ND (No Derivatives — cannot modify the asset)
  ❌ Stock with Standard Licence (typically prohibits digital distribution)
  ❌ Unknown / unverified licence
